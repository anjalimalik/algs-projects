
- To Run Project:

Use this command line to run the project:

$java -jar PlagiarismDetection.jar synonymsFilename.txt input1Filename.txt input2Filename.txt N

Example: 
$ java -jar PlagiarismDetection.jar syns.txt file1.txt file2.txt 3


- Assumptions about files: 

It is assumed that all files are given in correct format and words are separated by spaces.
